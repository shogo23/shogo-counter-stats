== Description ==
Track your visitors and monitor traffic and statistics of your website or blog.


== INSTALLATION ==
1. Clone/Download the plugin from https://github.com/shogo23/shogo-counter-stats to your wordpress plugin diretory. Make sure the
plugin directory's name is "shogo-counter-stats".

2. Login to your wordpress admin panel click plugins and activate Shogo Counter Stats.
